Scale Accurate 3D Model

  • User takes pictures of their space using ordinary smart-phone or DSLR camera
  • User uploads pictures to viscommerce.com site via Web-Browser or native clients
  • Using the pictures and an AI program MeshRoom a 3D model is reconstructed.
  • From reconstruced 3D model a scale-accurate CAD model is developed with Blender
  • Package up CAD model into “eSpace3D App” for user to download and use.

Scene Navigation

  • Zoom using mouse and keyboard keys (up, down).
  • Pan screen in ‘X’ and ‘Y’ directions.
  • Rotate model around ‘Y’ axis.
  • Reset scene to original camera settings.
  • Viewpoints let’s a user select from a pre-defined camera positions such as “front”, “back”, “left”, “right”, “top”.
  • Modelled on Unity3D and Blender.

Object Operations

  • Select an object at a time with mouse left button.
  • Deselect will deselect and remove highlight of selected object.
  • Delete will delete the selected object
  • Move, Rotate, Scale – As their names suggest they will make “Gizmos” visible that enable ‘move’, ‘rotate’ and ‘scale’ for selected object
  • Reset will remove highlight and gizmos from selected object.
  • Modelled on Unity3D and Blender
